# 🔫 러시안 룰렛
<br/>
<br/>

## 🎲 규칙
---
>- 게임에 참가할 인원과 총알 갯수를 입력받고
>- 총알을 랜덤으로 격발 시킨다.
>- 총알을 모두 소진하면 게임이 종료되고 생존자 명단을 출력한다.

<br/>
<br/>

## 🚀 기능 구현
---
1. 참가할 인원
    - 최소 2명 최대 10명의 인원까지 제한
    - 범위를 벗어나거나, 숫자가 아닌 값을 입력할 경우
    제대로 된 값이 들어올 때까지 입력받아야 한다.
2. 플레이어 이름 등록
    - 이름 글자 수를 제한 (최소 2글자 ~ 최대 10글자)
    - 어떠한 문자도 입력이 가능하지만 글자 수의 범위가 벗어난다면 제대로 된 값이 들어올 때까지 입력받아야 한다.
3. 총알 갯수 입력받기
    - 총알 갯수 제한 (최소 1발 ~ 최대 20발)
    - 범위를 벗어나거나, 숫자가 아닌 값을 입력할 경우
    제대로 된 값이 들어올 때까지 입력받아야 한다.
4. 랜덤 격발
    - (엔터를 입력받을 때마다) 난수를 생성하여 격발 확률 50%, 미격발 확률 50%로 설정.
5. 엔터로 격발 신호 입력받기
    - 어떠한 문자를 입력받아도 게임이 진행될 수 있어야 한다.
6. 결과
    - 생존자 명단을 출력
    - 생존자가 한 명일 경우 실탄의 갯수와 상관없이 게임은 종료된다.
